Mapping to GlyGen IDs
Note: Given a user-provided image of a glycan, this application outputs the glycan IDs of the five most similar glycans in the GlyGen database. The classification is based solely on the features that can be extracted from the images of the glycan structures. These features are extracted using the bagOfFeatures command in Matlab's Computer Vision toolbox. The features are used by the Neural Network algorithm from Matlab's Deep Learning Toolbox to identify the potential glycan ID of the image. This algorithm can only match images to glycan IDs that are contained in GlyGen.
